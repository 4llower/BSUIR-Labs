# Лабораторная работа №4

## Задание №1

~~~
Написать программу на языке C#, использующую возможности стандартных неуправляемых библиотек Windows. Желательно при этом не писать неуправляемый код самому.
~~~

* `Tetris.cs` (класс реализующий работу с GDI, простой тетрис на рабочем столе)
* `Program.cs` (осуществляет вызов класса Tetris.cs)

## Задание №2

~~~
Написать динамическую библиотеку на неуправляемом коде и программу на C#, использующую ее. Библиотека должна предоставлять функции с различными конвенциями вызова.
~~~

В данном задании реализовывал неявное дерево отрезков(`ImplicitTreeDllCode` сама реализация на языке C++), то есть структура позволяющая хранить отрезки очень большой длины. Структура полезна тем, что можно осуществлять быстрое добавление/удаление больших чисел, а также поиск количества больших чисел с L по R `(Пример: {1, 2, 3, 4, 5} количество элементов больших 1 и меньших 5 => 3)`.
`ImpclicitTree/Program.cs` -> тут находится взаимодействие с этой библиотекой.
